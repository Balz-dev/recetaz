# SEO — RecetaZ

Contexto: RecetaZ para médicos independientes en México.
Palabras clave: "recetas médicas", "médicos independientes", "México", "sin Word", "offline".

## Checklist Obligatorio por Página Nueva

- [ ] `<title>` descriptivo con palabra clave
- [ ] `<meta description>` de 155 caracteres
- [ ] Un solo `<h1>` con palabra clave
- [ ] HTML5 semántico (`<main>`, `<section>`, `<article>`)
- [ ] Open Graph tags (`og:title`, `og:description`, `og:image`)
- [ ] Schema markup JSON-LD (Organization + SoftwareApplication)
- [ ] Entrada en sitemap.xml

## Rastreo Google

**No bloquear**: `/`, `/demo`, `/precios`, `/faq`, landing pages
**Sí bloquear**: `/admin`, `/api`, rutas de desarrollo

## Prohibiciones

- ❌ No alterar branding/identidad
- ❌ No forzar keywords que suenen antinaturales
- ❌ No bloquear páginas importantes en robots.txt
- ❌ No usar técnicas black hat o spam de keywords
- ❌ No generar contenido de relleno

## Targets de Performance

- LCP < 2.5s
- FID < 100ms
- CLS < 0.1
